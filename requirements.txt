# requirements.txt — Automation Test Stability Analytics
#
# Install:
#   pip install -r requirements.txt
#
# Upgrade all pins at once (quarterly or before major releases):
#   pip install --upgrade -r requirements.txt
#   pip freeze > requirements.txt          # capture new pins
#   python -m pytest tests/               # verify nothing broke
#
# Why pins matter for this project
# ─────────────────────────────────
# FlagEmbedding / sentence-transformers change their pooling implementation
# across minor versions.  An unintentional upgrade can silently shift cosine
# similarities by 5–15 points, invalidating every cached embedding BLOB in
# the `embeddings` table.  Pin the model library and bump deliberately.
#
# When you do bump FlagEmbedding, run:
#   python pipeline.py --map-defects --force-reembed --overwrite-mappings
# to recompute all vectors under the new model version.
# ─────────────────────────────────────────────────────────────────────────────
